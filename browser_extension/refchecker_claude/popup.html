RefChecker

与桌面端一致：启用 LLM 后会先提取标题、作者、年份、DOI、URL；LLM 未解析出的字段会保留本地规则结果。大模型不判断文献真伪；API Key 在 RefChecker 桌面端或本地服务环境变量中配置，插件不保存 Key。
拖拽带数字的小标签调整优先级；单击小标签可加入/退出搜索链。Springer、IEEE、CORE 需要先配置 API Key。
打开 RefChecker 桌面版会自动启动本地服务；也可手动启动：python .\refchecker_http_server.py
推荐用法：链接问题请右键复制完整 URL/DOI，粘贴后点“核查粘贴链接”；选中文本只核查完整参考文献正文，不再自动识别 arXiv / PNAS 这类短标签或隐藏链接。
